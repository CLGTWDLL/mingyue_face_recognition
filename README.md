# mingyue_face_recognition
重庆大学24级明月科创实验班线性代数课程项目。人脸识别部分使用了PCA和SVM，PCA部分使用了小矩阵技巧和大矩阵特征向量转换以提高准确率。
操作说明：1，使用前下载MATLAB Support Package for USB Webcams工具箱，否则无法正常使用摄像头
